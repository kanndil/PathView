# ğŸ“– Overview

Extracts the paths generated by [OpenSTA](https://github.com/The-OpenROAD-Project/OpenSTA) reports and uses [netlistsvg](https://github.com/nturley/netlistsvg) to produce interactive SVG schematics for the extracted paths. The produced schematics interactively show the timing details of each cell for any standard cell library. Sky130_fd_sc_hd is supported as the default standard cell library, howeve, to add other libraries check [Support new library]().

# ğŸ” How to use

## First: Generate the timing report using [OpenSTA](https://github.com/The-OpenROAD-Project/OpenSTA)

It is necessary to include the following feilds in the report_checks command:

    -fields {slew cap input_pins nets fanout}

For example:

    report_checks -path_delay min_max -format full_clock_expanded -fields {slew cap input_pins nets fanout}  -endpoint_count 10 -unique_paths_to_endpoint > ./sta_report.rpt

<br/><br/>

## Second: Run interactive_SVG_schematics

    python3 interactive_SVG_schematics.py -i <staReportFile.txt>
